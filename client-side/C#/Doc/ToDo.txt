GlobeSpotter Window:
====================
  - Als je een GlobeSpotter window wordt geopend, dan moet deze initieel onderaan het scherm worden getoond i.p.v. naast de kaart.


Recent Recording Layer:
=======================
  - Bij het afsluiten van de laag hoeft geen melding te komen dat je deze laag wilt opslaan.
  - De limiet van 10 karakters per item in de database moet eruit.
  - De recent recording layer button ook in de layer tab van de map group plaatsen.
  - Bij PIP images moet ook nog de rotation toevoegen


Open Location Tool:
===================
  - De open locatie tool moet de focus verliezen als deze niet meer gebruikt kan worden.


Bugs in ArcGIS Pro / SDK:
=========================
  - Het is niet mogelijk om een laag te editen als deze initieel in een grouplayer geplaatst wordt.
  - Je kunt geen coordinaat transformatie uitvoeren naar 2 dezelfde coordinaat systemen, bijvoorbeeld van RD naar RD.
  - Als ik 2 keer een geoprocessing tool aanroep vanuit de code, dan gaat de 2e aanroep niet goed, terwijl ik wel netjes wacht tot de eerste is afgerond.
  - Dit voorbeeld om een default point symbol te maken werkt niet, elke keer krijgt deze een andere kleur:
		var pointSymbol = SymbolFactory.ConstructPointSymbol(ColorFactory.Blue, 7.0, SimpleMarkerStyle.Circle);
        var pointSymbolTemplate = pointSymbol.MakeSymbolReference();
        var uniqueValueRendererDefinition = new UniqueValueRendererDefinition(FieldNames, pointSymbolTemplate);
        var uniqueValueRenderer = (CIMUniqueValueRenderer)Layer.CreateRenderer(uniqueValueRendererDefinition);
        FeatureLayer.SetRenderer(uniqueValueRenderer);
